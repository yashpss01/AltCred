# AltCred Machine Learning Module

This directory contains the source code, models, and pipelines for the AltCred credit scoring machine learning models.

## 📂 Structure
- `models/`: Trained model binaries (e.g., `.pkl`, `.onnx`).
- `src/`: Core Python source code for data processing and inference.
- `pipeline/`: Reusable scripts for training and feature engineering pipelines.

## 🚀 Getting Started
1. Install Python 3.9+
2. Create a virtual environment: `python -m venv venv`
3. Install dependencies: `pip install -r requirements.txt`
